## 1. 개요

### 1.1 배경
- AI 에이전트 스킬이 많아지고 있지만, 발견하고 사용하기 어려움
- 개발자는 알아서 쓰지만, 비개발자(기획/디자인/QA)는 진입장벽 높음
- GitHub처럼 스킬을 공유하고 발견하는 허브가 필요

### 1.2 비전
**"AI 스킬의 GitHub — 누구나 쉽게 발견하고 사용"**

### 1.3 목표
- 비개발자도 쉽게 스킬 발견/사용
- 스킬 경쟁을 통한 품질 향상
- 오픈 커뮤니티 생태계 구축

### 1.4 비목표 (MVP 제외)
- 경제 레이어 (수익 모델)
- 모바일 앱
- Self-hosted 버전

## 2. 타겟 사용자

| 직군 | 니즈 | 예시 스킬 |
|------|------|----------|

### 2.2 Secondary: 개발자
- 알아서 잘 씀
- 파워유저, 스킬 제작자

## 3. 핵심 기능

### 3.1 UX 컨셉: 대화 + 스토어 하이브리드

```
┌─────────────────────────────────────┐
│  💬 대화창 (ChatGPT/Claude 스타일)   │
│  "기획서 작성 도와줄 스킬 찾아줘"      │
│  → AI가 맥락 파악 후 스킬 추천        │
├─────────────────────────────────────┤
│  🏪 스킬 스토어 (App Store 스타일)   │
│  카테고리별 브라우징, 검색, 설치      │
└─────────────────────────────────────┘
```

**핵심:** 비개발자도 자연어로 원하는 스킬 발견

### 3.2 스킬 마켓플레이스

#### 유저 스토리
```
As a 기획자
I want 대화로 스킬을 추천받거나, 스토어에서 직접 찾고 싶다
So that 내 상황에 맞는 최적의 스킬을 쉽게 발견할 수 있다

수용 기준:
- [ ] 대화형 스킬 추천 (자연어 입력)
- [ ] App Store 스타일 브라우징
- [ ] 직군별 카테고리
- [ ] 스킬 검색 (이름, 태그, 설명)
- [ ] 스킬 상세 페이지 (설명, 사용법, 평점)
- [ ] 원클릭 설치/사용
```

#### 기능 목록
- [ ] 💬 대화형 스킬 추천 AI
- [ ] 🏪 스토어 UI (카드형 스킬 목록)
- [ ] 직군별 카테고리
- [ ] 스킬 검색/필터
- [ ] 스킬 상세 페이지
- [ ] 인기/트렌딩 스킬
- [ ] 태그 시스템

### 3.2 스킬 등록

#### 유저 스토리
```
As a 스킬 제작자
I want GitHub repo에서 스킬을 등록하고 싶다
So that 다른 사람들이 내 스킬을 발견하고 사용할 수 있다

수용 기준:
- [ ] GitHub URL로 스킬 등록
- [ ] 자동 메타데이터 파싱 (frontmatter)
- [ ] 스킬 미리보기
- [ ] 버전 관리 (git tag 기반)
```

### 3.3 토너먼트 시스템

#### 유저 스토리
```
As a 사용자
I want 같은 목적의 스킬들이 경쟁하는 걸 보고 싶다
So that 어떤 스킬이 더 좋은지 객관적으로 알 수 있다

수용 기준:
- [ ] 카테고리별 토너먼트
- [ ] 1:1 스킬 대결
- [ ] 결과 기반 랭킹 업데이트
- [ ] 토너먼트 히스토리
```

### 3.4 랭킹/리더보드

#### 유저 스토리
```
As a 사용자
I want 가장 좋은 스킬이 뭔지 한눈에 보고 싶다
So that 검증된 스킬을 빠르게 선택할 수 있다

수용 기준:
- [ ] 전체 랭킹
- [ ] 카테고리별 랭킹
- [ ] ELO 레이팅 시스템
- [ ] 랭킹 변동 히스토리
```
