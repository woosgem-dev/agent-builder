  structure:
    header:
      - icon: string (emoji or image)
      - badge?: "Featured" | "New" | "Popular"
    
    body:
      - name: string (bold, truncate 1 line)
      - author: string (muted, link â†’ /authors/{id})
      - description: string (truncate 2 lines)
      - tags: Tag[] (max 3, overflow â†’ +N)
    
    footer:
      - rating: number (â˜… display)
      - installs: number (formatted: 1.2k)
      - updated?: relative time
  
  states:
    hover: elevation increase, border highlight
    focus: outline ring

### Component: SearchBar

```yaml
SearchBar:
  type: Input (controlled)
  
  structure:
    - leadingIcon: Search
    - input: text
    - trailingSlot: keyboard shortcut badge
    - dropdown: SearchSuggestions (on focus)
  
  behavior:
    - debounce: 300ms
    - minChars: 2
    - suggestions: recent + popular + matches
    - submit: navigate to /search?q={query}
