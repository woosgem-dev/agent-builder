# SkillHub - Project Brief (Draft v0.1)

> ⚠️ 초안입니다. Scouter, Planner, Senior들의 피드백을 받아 수정 예정.

## 비전

**"AI 스킬의 GitHub"** - AI 에이전트 스킬을 발견, 공유, 경쟁, 진화시키는 오픈 플랫폼

## 배경

- remote-skill-registry 프로젝트에서 스킬 토너먼트 시스템 POC 완료
- 커뮤니티 기반 스킬 생태계 필요성 확인
- GitHub가 코드의 허브가 된 것처럼, AI 스킬의 허브가 필요

## 핵심 가치

1. **오픈**: 누구나 스킬을 공유하고 발견할 수 있음
2. **경쟁**: 스킬끼리 토너먼트로 경쟁, 최고의 스킬이 살아남음
3. **진화**: 피드백 기반 자동 개선, 스킬이 스스로 성장
4. **비영리**: 커뮤니티 중심, 수익 모델은 당분간 배제
